TEMAS APRENDIDOS

En esta primera semana del curso he aprendido a utilizar Visual Studio Code, también he aprendido los conceptos básicos de HTML y cómo utilizar las abreviaturas de Emmet para acelerar la escritura de código HTML. He descubierto cómo estructurar una página web utilizando etiquetas como <html>, <head>, <body>, y cómo crear elementos como encabezados, párrafos y listas. Conclusión

En conclusión, durante esta primera parte del curso de desarrollo web, he tenido un gran aprendizaje que abarca una variedad de conceptos y tecnologías esenciales en la creación de sitios web interactivos y dinámicos. He adquirido habilidades fundamentales en HTML para estructurar el contenido, CSS para el diseño y la presentación, y JavaScript para la interacción y la lógica. Además, he aprendido a utilizar bibliotecas y herramientas populares como Bootstrap, Tailwind CSS y HTML canvas para mejorar la eficiencia y la estética de mis proyectos. A través del trabajo en clase, el estudio autodidacta, práctica activa y colaboración con otros estudiantes, he fortalecido mis habilidades y conocimientos en desarrollo web.

Recomendaciones

Recomendaciones

Aquí se presentan algunas recomendaciones para avanzar en el aprendizaje para el curso:

  • Crea proyectos personales, desafíos de código o contribuye a proyectos de código abierto relacionados con desarrollo web.

  • Aprovecha la abundante documentación en línea y tutoriales disponibles. Por ejemplo plataformas como MDN Web Docs, W3Schools y Stack Overflow

  • Participa en discusiones, comparte tus conocimientos y busca consejos. La colaboración con otros desarrolladores puede proporcionar perspectivas valiosas y soluciones

  • Mantente al día con las últimas tendencias, estándares y tecnologías. Sigue blogs, podcasts y conferencias relacionadas con el desarrollo web
